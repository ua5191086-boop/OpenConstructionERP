# ADR-007: Frontend: React 18 + TypeScript, Module Federation, own design system

**Status:** Accepted (2026-07-02) | **Decider:** Chief Architect / Product Owner
**Source:** SAD Tom 1, General Decision GD-007

## Decision
Frontend: React 18 + TypeScript, Module Federation, own design system.

## Rejected alternatives
Angular, Vue, Next.js as a hard requirement

## Rationale and consequences
Ecosystem and hiring; module federation allows independent delivery of domain modules. Heavy custom components: virtualized BOQ tables, canvas Gantt (50k activities), tunnel profile view.
