# ADR-013: AI layer: LLM-agnostic gateway (LiteLLM: Claude/GPT/local Llama via vLLM), LangGraph agents, hybrid RAG (pgvector + OpenSearch)

**Status:** Accepted (2026-07-02) | **Decider:** Chief Architect / Product Owner
**Source:** SAD Tom 1, General Decision GD-013

## Decision
AI layer: LLM-agnostic gateway (LiteLLM: Claude/GPT/local Llama via vLLM), LangGraph agents, hybrid RAG (pgvector + OpenSearch).

## Rejected alternatives
Hard dependency on a single LLM vendor

## Rationale and consequences
Government customers and banks will require on-premise inference. Human-confirmation principle: agents draft, humans approve any write to contractual/financial domains.
