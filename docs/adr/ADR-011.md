# ADR-011: Search: OpenSearch (BM25 + kNN hybrid for RAG)

**Status:** Accepted (2026-07-02) | **Decider:** Chief Architect / Product Owner
**Source:** SAD Tom 1, General Decision GD-011

## Decision
Search: OpenSearch (BM25 + kNN hybrid for RAG).

## Rejected alternatives
Elasticsearch (license), Meilisearch

## Rationale and consequences
License cleanliness for an open-source project; full-text over millions of documents plus vector retrieval for AI.
