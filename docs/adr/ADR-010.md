# ADR-010: File storage: MinIO (S3-compatible), versioning, WORM Object Lock for contractual documents

**Status:** Accepted (2026-07-02) | **Decider:** Chief Architect / Product Owner
**Source:** SAD Tom 1, General Decision GD-010

## Decision
File storage: MinIO (S3-compatible), versioning, WORM Object Lock for contractual documents.

## Rejected alternatives
Plain filesystem, GridFS

## Rationale and consequences
On-premise S3 plus legal immutability for Claims/Correspondence/Contracts.
