# ADR-009: AuthN/AuthZ: Keycloak (OIDC/SAML/LDAP) + OPA/Rego for ABAC policies

**Status:** Accepted (2026-07-02) | **Decider:** Chief Architect / Product Owner
**Source:** SAD Tom 1, General Decision GD-009

## Decision
AuthN/AuthZ: Keycloak (OIDC/SAML/LDAP) + OPA/Rego for ABAC policies.

## Rejected alternatives
Custom auth implementation

## Rationale and consequences
Banks and auditors require enterprise SSO from day one. Access = Role x Scope x Attributes x Object status. AI agents are principals of the same model with mandatory action logs.
