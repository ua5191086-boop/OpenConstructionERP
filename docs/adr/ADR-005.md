# ADR-005: Analytical database: ClickHouse (CDC via Debezium -> Kafka)

**Status:** Accepted (2026-07-02) | **Decider:** Chief Architect / Product Owner
**Source:** SAD Tom 1, General Decision GD-005

## Decision
Analytical database: ClickHouse (CDC via Debezium -> Kafka).

## Rejected alternatives
Druid, BigQuery, pure PostgreSQL for analytics

## Rationale and consequences
EVM/KPI/telemetry on a $500M project is billions of rows. ClickHouse is the de-facto open-source standard for this class.
