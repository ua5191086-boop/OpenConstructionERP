# ADR-015: Deployment: Kubernetes-first (Helm, ArgoCD) with a mandatory single-node Docker Compose profile

**Status:** Accepted (2026-07-02) | **Decider:** Chief Architect / Product Owner
**Source:** SAD Tom 1, General Decision GD-015

## Decision
Deployment: Kubernetes-first (Helm, ArgoCD) with a mandatory single-node Docker Compose profile.

## Rejected alternatives
Kubernetes-only

## Rationale and consequences
A construction site in Turkmenistan is not AWS. The full stack must run on one 64GB server.
