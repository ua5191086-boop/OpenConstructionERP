# ADR-014: Multi-tenancy: row-level security within an installation + separate installations per holding

**Status:** Accepted (2026-07-02) | **Decider:** Chief Architect / Product Owner
**Source:** SAD Tom 1, General Decision GD-014

## Decision
Multi-tenancy: row-level security within an installation + separate installations per holding.

## Rejected alternatives
Schema-per-tenant

## Rationale and consequences
An EPC holding = one installation with RLS-isolated projects. Competing contractors never share an installation.
