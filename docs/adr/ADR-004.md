# ADR-004: Primary database: PostgreSQL 16 with extensions — PostGIS, TimescaleDB, pgvector, Apache AGE, ltree

**Status:** Accepted (2026-07-02) | **Decider:** Chief Architect / Product Owner
**Source:** SAD Tom 1, General Decision GD-004

## Decision
Primary database: PostgreSQL 16 with extensions — PostGIS, TimescaleDB, pgvector, Apache AGE, ltree.

## Rejected alternatives
Oracle, MS SQL, MongoDB as primary; Neo4j as a separate graph DB

## Rationale and consequences
One DBMS covers relational, geo, time-series, vectors and graph. Radically simplifies on-premise operation for customers in Central Asia/CIS where clouds are unavailable. Neo4j removed: Apache AGE gives openCypher inside PostgreSQL.
