# ADR-008: Mobile: Flutter, offline-first (SQLite + event-log sync, CRDT conflict resolution)

**Status:** Accepted (2026-07-02) | **Decider:** Chief Architect / Product Owner
**Source:** SAD Tom 1, General Decision GD-008

## Decision
Mobile: Flutter, offline-first (SQLite + event-log sync, CRDT conflict resolution).

## Rejected alternatives
React Native (previous draft), PWA

## Rationale and consequences
A tunnel has no connectivity. Foreman, surveyor and HSE inspector work offline for days. Offline-first is a survival condition on site, not a feature.
