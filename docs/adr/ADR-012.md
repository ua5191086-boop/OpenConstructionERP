# ADR-012: BIM: full OpenBIM — IfcOpenShell (IFC 4.3 incl. infrastructure entities) + xeokit viewer + own BCF 3.0 server

**Status:** Accepted (2026-07-02) | **Decider:** Chief Architect / Product Owner
**Source:** SAD Tom 1, General Decision GD-012

## Decision
BIM: full OpenBIM — IfcOpenShell (IFC 4.3 incl. infrastructure entities) + xeokit viewer + own BCF 3.0 server.

## Rejected alternatives
Autodesk Forge / Bentley APIs as the core (vendor lock-in)

## Rationale and consequences
No dependency on Autodesk/Bentley for the core path. Geometry fragmented into XKT tiles in MinIO; model elements linked to ontology objects, schedule activities (4D) and BOQ items (5D).
