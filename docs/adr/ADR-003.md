# ADR-003: Backend languages: Go (core/services) + Python (AI, analytics, BIM processing) + TypeScript (BFF/gateway)

**Status:** Accepted (2026-07-02) | **Decider:** Chief Architect / Product Owner
**Source:** SAD Tom 1, General Decision GD-003

## Decision
Backend languages: Go (core/services) + Python (AI, analytics, BIM processing) + TypeScript (BFF/gateway).

## Rejected alternatives
Java/Spring, C#/.NET, Rust, NestJS as primary backend

## Rationale and consequences
Go: performance, simple hiring, low entry barrier for open-source contributors. Python is unavoidable for AI and IfcOpenShell. Rust and NestJS removed from the declared stack — four backend languages is an anti-pattern for this team size.
